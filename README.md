#  cmake requirement
To play, clone the repository and navigate to the game directory:

# 1. Clone the repository:
    git clone [repository_url]


# 2. Navigate to the game directory and build the project:
   
    cd [directory_name]

    mkdir build && cd build

    cmake .. && make


# 3. Run the game:
    ./GAME
